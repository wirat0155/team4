เครื่องมือในการพัฒนาระบบ(Tools) : Visual Studio Code เพราะเป็น Code editer ที่ดีมากๆ ตัวนึง มีExtension ที่รองรับการเขียนโปรแกรมได้หลายภาษา สามารถปรับสี ขนาด ของตัวอักษรได้ 

เทคนิค (Techniques) : การเขียน Use Case Diagram เป็นเทคนิคที่สามารถมองเห็นภาพรวมของตัวระบบได้ง่ายขึ้น เพราะตัว Use Case Diagram เป็น Diagram ที่มีฟังก์ชันของระบบทั้งหมดรวมอยู่ในที่เดียว สามารถคาดคะเนหน้าตาของระบบได้อย่างง่ายดาย
